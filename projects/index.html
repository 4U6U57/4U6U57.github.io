This Site

This site definitely counts as one of my more ambitious projects.

Rather than use a hosting service such as Wordpress, Google Sites, etc. as I've done in the past, I decided to try my hand at writing base HTML on an actual server.

It started out in my uni's UNIX Timeshare, which happened to have a public_html/ directory for posting basic sites.

It was slow at first, with me learning very basic HTML/CSS and Javascript through Codeacademy.

It was also around this time that I discovered the Polymer Project, a web component library by Google incorperating ideals of Material Design.

Currently, I am dual hosting on the UCSC server, as well as GitHub pages, so I can make use of GitHub's versioning.

I'm also using it to host papers that I write for school, so that they will be added to Google Scholar.
