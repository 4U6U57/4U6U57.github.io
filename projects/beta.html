Personal Website

Programming in Polymer

This site definitely counts as one of my more ambitious projects. Rather than use a hosting service such as Wordpress, Google Sites, etc. as I've done in the past, I decided to try my hand at writing base HTML on an actual server. It started out on my uni's UNIX Timeshare, which happened to have a public_html/ directory for posting raw HTML files. It was slow at first, with me learning very basic HTML/CSS and Javascript through Codeacademy. Around this time, I discovered the Polymer Project, a web component library by Google incorparating Material Design elements. I instantly adapted it.

Currently, I am dual hosting on the UCSC server, as well as GitHub pages, so I can make use of GitHub's version control and keep track of my progress Just recently, I added the Publications page so that I can publish my school papers online and have them parsed by Google Scholar. I am most recently working on the this page, the Projects section.
