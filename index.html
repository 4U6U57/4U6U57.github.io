This Site

I am currently learning HTML/CSS/JS/jQ, so this site is some good practice.

It is hosted on GitHub Pages (github.io), and is mirrored on the UCSC public server (people.ucsc.edu).

The project is completely open source, and the source code (github.com) is made available to the public.

It is primarily built on web components published by the Polymer Project (Google I/O 2014).
